## About This Repository

**CAT Email Agent** solves the email privacy and automation problem.

### 🚨 The Problem
- ❌ Email clients store everything forever
- ❌ Your email is tracked and profiled for profit  
- ❌ Third parties get access to your inbox
- ❌ No control over data collection
- ❌ Spam and repetitive tasks waste hours

### ✅ Our Solution
- 🔒 **Zero data retention** - in-memory processing only
- 🚫 **No profiling** - no tracking, no behavior analysis
- 🛡️ **No sharing** - blocked by design, never sent to third parties
- 👤 **You control it** - OAuth credentials only you provide
- 🤖 **AI automation** - natural language commands
- ⚡ **5-minute setup** - automated installation
- 📖 **Open source** - fully transparent, auditable code
- 🎯 **Compliant** - NZ Privacy Act 2020 (all 13 principles)

### 🚀 Quick Start
```bash
# Linux/macOS
bash install.sh

# Windows  
powershell -ExecutionPolicy Bypass -File install.ps1

# Then configure and run
npm run dev "delete spam emails"
```
