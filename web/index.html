Quick Install

Linux/macOS

git clone https://github.com/coastalalpine/cat-mail.git
cd cat-mail
bash install.sh

Windows

git clone https://github.com/coastalalpine/cat-mail.git
cd cat-mail
powershell -ExecutionPolicy Bypass -File install.ps1

Then configure your credentials in .env and start using it!
